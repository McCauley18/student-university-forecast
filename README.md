# student-university-forecast
A machine learning application that predicts whether a student is likely to attend university based on academic, demographic, and socio-economic factors. Built with Python and scikit-learn, this project demonstrates data preprocessing, feature engineering, model training, and evaluation to forecast student university enrollment outcomes.

# Features
Predict student college enrollment with high accuracy.
Preprocessing pipeline for data transformation.
Uses a trained ML model with a saved scaler.
Interactive Streamlit dashboard for easy user input and visualization.

# Installation
1. git clone https://github.com/McCauley18/student-university-forecast.git
  cd student-university-forecast

2. Create a virtual environment
  python -m venv venv

3. Activate the virtual environment
   Windows: venv\Scripts\activate
   macOS/Linux: source venv/bin/activate

4. pip install -r requirements.txt
   Install dependencies

5. Run app locally
   streamlit run app.py
